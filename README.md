# Lead Scoring Demo

This repository contains a simple Python-based lead scoring.

## Files
- `lead_scoring.py` – Python script that assigns scores to leads based on predefined rules.
- `lead_scores.csv` – Output file containing lead names and their calculated scores.

## Description
The script processes a small dataset of startup leads and assigns scores based on factors such as role, industry, country, funding stage, and prior engagement.

The output is generated programmatically and saved in CSV format for easy verification.
